Run the ORBWEVA Pitch Night skill (`skills/pitch-night/SKILL.md`), Stage 1.

Build the spoken 3-minute pitch script for the current venture:

1. Gather the venture basics — what it is, who it's for, the proof you have, the channel, the market.
2. Write the script to the second-budgeted structure in the skill (hook → problem made foreign-legible → product → proof → go-to-market → market size + ask), ~390–450 words for a 3-minute slot.
3. One claim per sentence; replace every adjective with a number or a proper noun; cut anything the judge already knows.
4. Flag every place the founder must insert a real number (proof, market size) — do not invent figures.

Then offer `/pitch-night:translate` (make the home market legible) and `/pitch-night:score`. Reference the full SKILL.md for the structure, rules, and word budget.
