Run the ORBWEVA Pitch Night skill (`skills/pitch-night/SKILL.md`), Stage 3.

Take the current pitch and produce a rehearsal plan:

1. Time it out loud against a real timer — page-time lies, mouth-time is truth. If it runs past the warning, it's over; cut, don't speed up.
2. Trim to land ~10 seconds early, ending on the ask with breathing room.
3. Drill the first 20 seconds until the hook is automatic.
4. Prepare a 90-second fallback (hook → problem → proof → ask) for overruns or Q&A.
5. Strip filler and hedges; plan for a broken demo (one screenshot + one sentence that carry the product).
6. If virtual, speak to the camera and the mic, not the gallery.

Reference the full SKILL.md for the delivery checklist.
